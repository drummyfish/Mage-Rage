Mage Rage

2013

This is a PC game developed by Miloslav "tastyfish" Číž with a little help of
Martin Gabriel for devbook.cz's game contest Magické léto 2013. It is free for
everyone to play as well as to modify it as long as we'll be mentioned as the
authors of the original game. The game hides nothing, it's source codes for
MS Visual C++ 2010 and it's resources are available for anyone to download,
modify and distribute.

Author of everything, that means the concept, script, level design, source code
